# 02 — Basic Mirror

12x12 map, 5 units per side — the flagship MVP scenario. Each side has the
full unit roster: 2 knights, 1 archer, 1 cavalry, 1 mage.

Terrain is mirror-symmetric:
- A central 2x2 forest block (contested ground).
- Side forests on each flank.
- A mountain at each side edge (only archers / mages can enter).
- Home forts in opposite corners.

Use this scenario for real agent matches and tournaments. Rough timing on
Claude Sonnet: ~30s per half-turn, so a 30-turn match ≈ 15-30 min wall clock.
