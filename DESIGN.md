You are an experienced game designer that is proficient in designing interesting

The games in the current market are all designed for humans. With AI agents
being smart enough and can reason and act, I think we can probably design a games
that include AI agents that work very differently from the current games and
therefore introduce a very different dynamics. I have a few thoughts and I want
you to help me design such games. The following points are a brain dump from
what I was thinking and by no means represent a mature idea on the game design.

1. NPCs of current games are mostly policy based. The NPCs actions are
   pre-determined and have little flexibility and therefore limits the space of
   possible game experience. One possibility is to actually redesign the whole
   NPCs of games to be more autonomous and can reason and act based on the
   interaction with the users.
2. We can also think about designing a game that is completely played by agents.
   Traditional games played by humans are based on monitor and keyboard and mouse.
   I think we should be able to design a game where agents interact with the
   game completely in MCP or CLI, with a minimal UI that just conveys what happens
   to the human. Agents get the status of the game via MCP/CLI, and then reasons,
   and then acts with MCP/CLI. The MCP server is the ultimate arbiter of the
   state of the game. And we can potentially have many agents connecting to the
   MCP server to play together or compete with each other.
3. For the game in point 2, it is also possible to allow human to participate in
   the game but not directly. For example, human might form a team with the agent,
   proposes and discusses with the agent on the game strategy. I haven't decided
   whether we should allow human to directly participate in the game.

What do you think about this idea? What are some of the games currently available
that you think we can reference to design such a game that is primarily played
by AI agents?
